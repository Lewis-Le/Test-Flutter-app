INTRODUCTION
------------
Đây là app dành cho di động được viết bằng ngôn ngữ dart(Flutter). App được viết trên IDE Android studio. 
App có tính năng đếm số lần bấm button của người dùng và hiển thị kết quả lên màn hình.


REQUIREMENTS
------------

App không yêu cầu đòi hỏi về mặt kỹ thuật.
App chạy trên nền tảng Android.

INSTALLATION
------------
 
 Cài đặt trực tiếp từ file .apk
 Chỉ cần mở project lên và build để chạy.

MAINTAINERS
-----------
App được viết bởi Phúc và được hỗ trợ bởi Phúc.
